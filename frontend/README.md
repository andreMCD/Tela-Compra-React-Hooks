# Teste Prático **Agência N1** Tela de Compra / Sucesso da Compra

## Instalação

Faça o clone e utilize os comandos:

```

$ git clone *link do repositório*
$ cd *caminho do diretório*
$ npm install
$ npm start

```

Acesse a pagina no link: http://localhost:3000
